# AIDE-HsswSerial
Arduino IDE HsswSerial library

- Arduino IDE library for Arduino UNO
	- Place "HsswSerial/" directory in Arduino IDE's "\<Documents\>/Arduino/libraries/" directory.
- See also link below (Japanese page):
